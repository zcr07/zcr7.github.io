打开Firefox 输入   about:profiles

## 2. 在桌面创建快捷方式


右键/新建快捷方式 / 请键入对象的位置(D) / 输入

"C:\3\firefox\firefox.exe" -no-remote -P "kevinzhcr"
或
"C:\3\firefox\firefox.exe" -no-remote -P "zb9678"

下一步/ 键入该快捷方式的名称(I): / kevinzhcr  /  完成

现在桌面上多了2个Firefox不同帐号的启动快捷方式
