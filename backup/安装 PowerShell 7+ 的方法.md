## 安装 PowerShell 7+ 的方法

方法一：用 Winget（推荐，最简单）

如果你的 Windows 里已经有 winget 包管理器（Win10 / Win11 默认带有）：
在命令提示符或 PowerShell 输入：

```

winget install --id Microsoft.Powershell --source winget

```
