## 热字串

- 每个热字串缩写的长度不能超过 40 个字符. 超出此长度后程序会向您提出警告. 

### 空格或 tab

- 要在替换文本后发送额外的空格或 tab, 可以把它们加在替换文本后, 但需要在末尾加上重音符/反引号(`). 

```
:*:bbb::By the way	`
```
注：如果没有 ` 这个符号，则不会显示空格或Tab.

### Tab键激活替换
- 当您输入"ab"后跟着 tab 时会触发热字串

```
:*:ab`t::By the way
```

### Enter键激活替换


```
:*:ab`n::By the way
```

###  , 键激活替换

```
:*:ab`,::By the way
```

### 空格在字符最前和最后的区别

- 在前  会正常输出空格
- 在后  则不会输出空格

```
::ttt:: by the way

::bbb::by the way 
```

## 超长替换

```
::bbb::
(
What happened?
123
)
``
